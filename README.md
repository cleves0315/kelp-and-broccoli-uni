## 介绍

海带与西兰花是一款计划管理工具，他界面简洁而又不失优雅，操作丝滑简单，让你每日的计划管理更加的简单。

## 功能

- 添加计划，输入计划名称
- 方便快速设定截止日期与重复周期
- 支持定时推送消息
- 计划类型分类（”我的一天“ 为当天主要任务）
